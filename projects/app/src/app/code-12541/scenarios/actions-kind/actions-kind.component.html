Cenario: Propriedade kind das Acoes
Valida como o kind de cada acao se comporta em cada header type.
Primary - Acoes com kind explicito (kind ignorado no primary default)
No primary default: primeira acao = primary, demais = secondary. Kind do PoPageAction NAO tem efeito.
Secondary - Acoes com kind explicito
No secondary default: primeira acao = primary (padrao do layout default), demais usam kind ou fallback secondary.
Tertiary - Acoes com kind explicito
No tertiary default: primeira acao = primary (padrao do layout default), demais usam kind ou fallback secondary.
Secondary - Acoes SEM kind (fallback)
Sem kind: primeira acao = primary, demais = secondary (fallback padrao).
Tertiary - Acoes com kind misto
Primeira = primary, segunda sem kind (secondary), terceira = tertiary explicito.
Secondary Mixed Layout - kind no primeiro botao
Mixed: primeiro botao usa action.kind || 'secondary'. Demais no dropdown.
Tertiary Mixed Layout - kind no primeiro botao
Mixed: primeiro botao usa action.kind || 'secondary'. Demais no dropdown.
