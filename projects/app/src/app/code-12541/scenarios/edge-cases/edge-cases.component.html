Cenario: Casos Limite (Edge Cases)
1. Subtitle sem Title (subtitle nao deve aparecer)
Subtitle fornecido mas sem title. O subtitle NAO deve ser renderizado.
2. Secondary sem titulo, sem acoes (header visivel pelo botao voltar)
Secondary sem titulo e sem acoes. Header deve ser visivel pelo botao voltar.
3. Tertiary sem titulo e sem acoes (header NAO deve aparecer)
Tertiary sem titulo e sem acoes. hasPageHeader() retorna false, header NAO visivel.
4. Primary sem titulo, sem acoes, sem breadcrumb (header NAO deve aparecer)
Primary sem nada. hasPageHeader() retorna false.
5. Breadcrumb no Secondary (breadcrumb ignorado)
Breadcrumb fornecido mas ignorado no secondary. Nao deve renderizar breadcrumb.
6. Breadcrumb no Tertiary (breadcrumb ignorado, sem espaco)
Breadcrumb fornecido mas ignorado no tertiary. Nao deve renderizar breadcrumb nem reservar espaco.
7. Header type invalido (deve usar primary como default)
Header type invalido. Deve fazer fallback para primary.
8. Actions layout invalido (deve usar default)
Actions layout invalido. Deve fazer fallback para default.
9. Apenas breadcrumb sem titulo (Primary)
Apenas breadcrumb, sem titulo. Header visivel pelo breadcrumb. Nao deve haver gap desnecessario.
10. Secondary subtitle sem title
Secondary com subtitle mas sem title. Subtitle NAO deve ser renderizado.
