Cenario: Comparacao dos 3 Header Types

Todos com as mesmas 3 acoes, titulo, subtitulo e breadcrumb (quando compativel).

1. Primary (padrao)

Conteudo do header primary. Breadcrumb visivel, titulo h2, primeira acao primary kind.

2. Secondary

Conteudo do header secondary. Breadcrumb NAO visivel, botao voltar, titulo h3 semibold, primeira acao primary kind no default layout.

3. Tertiary

Conteudo do header tertiary. Sem breadcrumb, sem botao voltar, titulo h3 regular, primeira acao primary kind no default layout.
