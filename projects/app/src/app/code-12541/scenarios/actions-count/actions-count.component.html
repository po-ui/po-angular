Cenario: Contagem de Acoes

Valida comportamento com diferentes quantidades de acoes em cada header type.

Primary - 0 acoes

Sem acoes - header visivel apenas pelo titulo.

Primary - 1 acao com icone

1 acao com icone, kind primary.

Primary - 1 acao sem icone

1 acao sem icone - label deve ser exibido em todas as telas.

Primary - 2 acoes

2 acoes: primeira primary, segunda secondary.

Primary - 3 acoes

3 acoes: primeira primary, demais secondary.

Primary - 5 acoes (overflow)

5 acoes: overflow, dropdown automatico.

Secondary - 0 acoes

Secondary sem acoes - header visivel pelo botao voltar.

Secondary - 3 acoes

Secondary com 3 acoes no default layout.

Tertiary - 0 acoes

Tertiary sem acoes - header visivel apenas pelo titulo.

Tertiary - 3 acoes

Tertiary com 3 acoes no default layout.
