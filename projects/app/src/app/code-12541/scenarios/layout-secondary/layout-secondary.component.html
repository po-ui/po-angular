Cenario: Secondary Header - Todos os Layouts

Header type secondary com 5 acoes. Botao voltar sempre presente.

1. Layout Default

Layout default no secondary: primeira acao primary kind, demais secondary. Dropdown para overflow.

2. Layout Dropdown

Layout dropdown no secondary: todas as acoes no dropdown. Botao voltar visivel.

3. Layout Mixed

Layout mixed no secondary: primeiro botao visivel (kind conforme action.kind), demais no dropdown.
