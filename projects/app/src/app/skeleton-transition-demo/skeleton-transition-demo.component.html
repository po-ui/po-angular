Demonstração Interativa dos Tokens de Transição

Clique no botão abaixo para alternar entre os tipos Normal (cinza claro) e Primary (cinza médio) do skeleton.

Edite os valores dos tokens CSS no arquivo CSS para testar diferentes configurações de transição e veja as mudanças em tempo real!

Skeleton com Tokens Customizáveis

Os tokens abaixo estão aplicados via :root no CSS. Altere os valores para testar!

Tokens CSS Ativos

Estes são os tokens que você pode editar no arquivo CSS:

Token CSS Valor Atual Descrição
--transition-duration 5s Duração da transição ao mudar de tipo (clique no botão)
--animation-duration 3s Duração das animações pulse/shimmer (ative animation="pulse" ou "shimmer")
--transition-property background-color Propriedade que sofre transição (teste: all, background, opacity)
--transition-timing ease-in-out Função de temporização (afeta transição E animações)

💡 Sugestões de Testes

Para ver diferenças dramáticas:
  • Super Lento: --transition-duration: 10s
  • Super Rápido: --transition-duration: 0.05s
  • Efeito Bounce: --transition-timing: cubic-bezier(0.68, -0.55, 0.27, 1.55)
  • Linear: --transition-timing: linear
  • Todas as propriedades: --transition-property: all
Como testar:
  1. Abra o arquivo skeleton-transition-demo.component.css
  2. Edite os valores dentro de :root
  3. Salve o arquivo
  4. Clique no botão "Alternar Tipo" para ver a transição em ação
