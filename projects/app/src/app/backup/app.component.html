2. Tipos de Skeleton - Cores Diferentes (p-type)

A propriedade p-type altera a cor de fundo do skeleton.

Type: Normal (Padrão)

Cor neutra clara - #D0D0D7

Type: Primary

Cor neutra média - #9191A1

Type: Content

Fundo branco - #FFF
